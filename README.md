# 📝 Todo - 待办任务管理系统

这是一个简洁高效的待办事项管理应用，旨在帮助用户更好地规划日常任务与时间。

## 📂 项目结构

本项目采用前后端分离架构：

- **`frontend/`**: 前端页面与交互逻辑代码。
- **`system/`**: 后端核心业务逻辑与接口处理。
- **`todo.sql`**: 数据库初始化脚本，包含表结构与基础数据。

## 🚀 快速开始

### 1. 环境准备
请确保你的电脑上已安装以下环境：
- JDK (如果是Java项目) / Node.js (如果是Node项目)
- MySQL 数据库

### 2. 数据库配置
1. 创建一个新的数据库（例如命名为 `todo_db`）。
2. 运行根目录下的 `todo.sql` 文件导入数据表。

### 3. 启动项目
- **后端**：进入 `system` 目录，配置数据库连接信息后启动服务。
- **前端**：进入 `frontend` 目录，安装依赖并启动开发服务器。
